# API tests samples

## Overview

Sample of one API test creating by Jest framework and Supertest library

## Running test

- `git clone https://github.com/loboykois/api_tests_samples.git`
- `npm install --save-dev jest`
- `npm install supertest --save-dev`
- `npm test`
